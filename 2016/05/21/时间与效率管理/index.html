时间与效率管理

发表于 2016-05-21   |   分类于 编程之路   |     |   本文总阅读量 次

把握好时间与效率在我的学习与计划中非常重要，通过有效的工具管理，不但能提高自己的效率和时间利用率，还能分析出改进的方法，不断的使得自己的时间利用最大化，效率最高。下面介绍我常用的工具。

番茄土豆（pomotodo）

基于番茄工作法和任务管理，番茄土豆帮助你更高效地完成工作

这是番茄土豆官网给出的介绍。事实上番茄工作法是简单易行的时间管理方法，是由弗朗西斯科·西里洛于1992年创立的一种相对于GTD更微观的时间管理方法。使用番茄工作法，选择一个待完成的任务，将番茄时间设为25分钟，专注工作，中途不允许做任何与该任务无关的事，直到番茄时钟响起，然后在纸上画一个X短暂休息一下（5分钟就行），每4个番茄时段多休息一会儿。番茄工作法极大地提高了工作的效率，还会有意想不到的成就感。

而番茄土豆这在此基础上加上了任务管理。成为一个完整的工作流效率工具,从收集想法、规划任务、专注工作、归纳分析，番茄土豆提供了完整的工作流效率管理。

从倒数第二张可以看到我在13-15日内完成的一些任务情况，通过这个工具我们能记住自己每天干了什么。而最后一张图这是给出了一周的分析图，用过分析数据，我发现自己早上的时间利用率可谓是极低，那么我便针对此专门做调整，加强早上的时间学习与利用。
除此之外，还能给自己添加目标，并将其嵌入到日历中，不断提醒自己。

市面上这类工具很多，但做的最好的就是这款pomotodo，无论是全平台（甚至有appwatch和Androidwear版本），还是良心到极致的付费，只要你想随时可以退款。强烈推荐有需求的朋友使用。

RescueTime

一款时间记录软件，手机和电脑都可使用，两者结合绝对完美！（我是这么认为的）

这款软件只需要在最初的几天做好设置，接下来会自动进行统计记录。它会在后台自动追踪你的使用时间，你是在玩微信还是在打电话还是在用电脑上网亦或是开着word在工作，它都能帮你监控哦！！

最后还会帮你进行统计，计算你的效率（浪费的时间、积极工作或者学习的时间占总时间的百分比和具体数值）。

最让我欣赏的是它能同时多平台使用（手机和电脑一起开着）不过你时间利用的大方向是绝对没错的！

工具是死的，人才是活的。如果你只是去使用工具，而不去做改变，那么以上的推荐并没有任何意义。
相反活用工具，通过工具不断的改变自己，使自己不断提高，才是工具最大的作用。
